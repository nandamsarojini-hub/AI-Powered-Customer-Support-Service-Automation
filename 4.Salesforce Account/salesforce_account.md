# Salesforce Developer Environment Setup

## 1. Introduction
To build a custom AI solution utilizing Prompt Builder and Agentforce, a dedicated Salesforce Developer Edition organization is required. This environment provides full access to enterprise-level customization tools, apex coding, and automation features.

## 2. Account Creation Steps
1. Navigate to the official developer portal: [Salesforce Developer Signup](https://developer.salesforce.com/signup).
2. Complete the registration form with the following details:
   * **First Name & Last Name:** Your Name
   * **Email Address:** Your primary email
   * **Role:** Developer
   * **Company:** Your College or Organization Name
   * **Country:** India
   * **Postal Code:** Local PIN code
   * **Username:** A globally unique identifier in email format (e.g., `developer@aisupport.com`).
3. Click **Sign me up**.

## 3. Activation and Initialization
1. Check the provided email inbox for the Salesforce activation link (may take 5–10 minutes).
2. Click **Verify Account**.
3. Set a secure password and establish a security question for account recovery.
4. Upon saving, the system automatically redirects to the **Salesforce Setup Home**, serving as the launchpad for all subsequent object and AI configurations.
