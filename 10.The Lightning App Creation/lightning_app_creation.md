# Lightning App Creation

## Overview
This document outlines the configuration of a custom Salesforce Lightning App. This application serves as the primary workspace for support agents, bringing together all the necessary custom objects, reports, and dashboards into a single, unified interface.

---

## Activity 1: Create a Lightning App

### Step 1: App Details and Branding
To initiate the app creation, navigate to the **Setup** page, search for "App Manager" in the Quick Find box, and select it. From the App Manager screen, click on **New Lightning App** as shown in image_7f9824.jpg.
*   **App Name:** AI-Powered Customer Support & Service Automation (image_7f9824.jpg).
*   **Developer Name:** Auto-populates based on the App Name (image_7f9824.jpg).
*   **Description:** Enter a meaningful description for the application (image_7f9824.jpg).
*   **Image & Branding:** Upload an optional logo and retain the default primary color hex value (image_7f9824.jpg, image_7f98dd.png).

### Step 2: App Options & Utility Items
*   Click **Next** through the App Options page, keeping the default standard navigation settings (image_7f98dd.png).
*   Click **Next** through the Utility Items page, leaving it as default (image_7f98dd.png).

### Step 3: Navigation Items
Select the items that will appear as tabs at the top of the application. Search for the custom objects created in previous steps and use the arrow buttons to move them into the **Selected Items** list (image_7f98dd.png). 
*   **Selected Items:** Customers, merchandises, Orders, User Cases, Knowledges, Reports, Dashboards (image_7f98dd.png, image_7f9c25.png).

### Step 4: User Profiles
Define which user profiles will have access to this specific application. 
*   Search for and select **System Administrator** and **Einstein Agent User** from the Available Profiles list and move them to the Selected Profiles list (image_7f98dd.png).

### Step 5: Save & Verify
*   Click **Save & Finish** (image_7f98dd.png).
*   Navigate to the App Launcher (the nine-dot grid icon) and open the "AI-Powered Customer Support & Service Automation" app. 
*   Verify that the app successfully loads with the designated navigation tabs across the top (Customers, merchandises, Orders, User Cases, Knowledges, Reports, Dashboards) and displays the corresponding list views correctly (image_7f9c25.png).
